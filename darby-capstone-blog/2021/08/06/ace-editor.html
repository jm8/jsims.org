A Better Code Editor

Published on Aug 6, 2021

    Before in our editor we just had a textarea. These are builtin to HTML, and it was okay to use them for starting out. However, textareas are not very good to use for writing code. For example, you can’t press tab to indent.

    I want a better code editor. The nice thing about programming is that you can often use other people’s code that they publish as open source. This way, I can use a better code text editor without having to make it myself (since that is not the main point of what I am trying to do).

    Also, since I am using React, I can use a component. So if I want to use (Ace Editor)[https://ace.c9.io/], all I have to do is install it

    npm install --save react-ace

    and change

    <textarea value={code} onChange={(e) => setCode(e.target.value)} />

    to

    <AceEditor value={code} onChange={(x) => setCode(x)}></AceEditor>
