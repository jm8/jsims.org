Making a programming language

    About me

    Hello, my name is Josh. My hobbies include programming and playing the cello. At Darby I am in the orchestra, Mathletes, and NHS, and cofounder of the computer club. I also like science fiction televison shows.

    My project

    My freshmen year, I took the AP Computer Science Principles class. The class was “language agnostic”, meaning that schools could chose any programming language they wanted to teach in during the class. However, they had to make up their own fake pseudocode language to use on the exam, which is described on the reference sheet. The thing is, that there’s no way to actually run this language.

    My project is going to be to program an interpreter for the AP CS Principles language. I think this will be useful because it will let future students learn that language better by writing programs in it. It will also help me develop my programming skills.
